# Project Foundation MOC

This Map of Content (MOC) provides links to the high-level, foundational markdown files that define the vision, architecture, and core principles of the entire LifeOS project.

## 1. Core Philosophy & Vision

These documents describe the "why" behind LifeOS, its goals, and the methodologies it is built upon.

- **[[../../Guiding-Principles.md|Guiding Principles]]**: The core values and philosophy driving the project.
- **[[../../docs/01_LifeOS_Master_Plan.md|LifeOS Master Plan]]**: The highest-level strategic plan and vision for the project.
- **[[../../docs/06_Personal_Growth_System.md|Personal Growth System]]**: The framework for how LifeOS facilitates personal development.
- **[[../../docs/07_Scientific_Foundation.md|Scientific Foundation]]**: The research and scientific principles underpinning the system.
- **[[../../Developer_Handbook_PARA+GTD+Zettelkasten_System.md|Developer Handbook (PARA/GTD/Zettelkasten)]]**: A guide to the information management methodologies used.

## 2. Architecture & Technical Strategy

These documents outline the technical "how" of the project, including architecture, data schemas, and standards.

- **[[../../docs/03_Architecture_Decisions.md|Architecture Decisions]]**: Records of significant architectural choices and their rationale.
- **[[../../Comprehensive-Tech-Strategy.md|Comprehensive Tech Strategy]]**: The overarching technical strategy for the project.
- **[[../../docs/02_Schema_and_Standards.md|Schema and Standards]]**: Definitions for data schemas and other project-wide standards.

## 3. AI Integration & Collaboration

This section covers how AI agents are integrated into the project and the guidelines for their operation.

- **[[../../docs/04_Agent_Guide.md|Agent Guide]]**: The primary guide for AI agents, detailing workflows and procedures.
- **[[../../docs/05_Master_System_Prompt.md|Master System Prompt]]**: The core identity and instruction set for the LifeOS AI.
- **[[../AI_Interaction_Guide.md|AI Interaction Guide]]**: Guidelines for how the user and AI should interact effectively.
- **[[../LifeOS_AI_Integration_Guide.md|LifeOS AI Integration Guide]]**: Specifics on how the AI is integrated into the LifeOS ecosystem.
