# Weekly Review Checklist

**Recommended Time:** Sunday evening, 60-75 minutes  
**Goal:** Get Clear, Get Current, Get Creative

---

## 1. Get Clear (30 min)

### Process All Inboxes
- [ ] Process all items in `GTD-Tasks/0-Inbox/`
- [ ] Process physical inbox (mail, papers on desk)
- [ ] Process email inbox to zero
- [ ] Process phone notifications and messages
- [ ] Process any loose notes or voice memos

### Review Waiting-For
- [ ] Review `GTD-Tasks/2-Waiting-For/`
- [ ] Chase any items that are overdue
- [ ] Update status on delegated items

### Process Notes
- [ ] Review `GTD-Tasks/6-Needs-Processing/`
- [ ] Process 2-3 notes into permanent notes or tasks
- [ ] File or delete fleeting notes

---

## 2. Get Current (20 min)

### Review Projects
- [ ] Review all `Projects/Active/`
- [ ] Update project status
- [ ] Ensure each active project has at least one next action
- [ ] Archive any completed projects

### Review Areas
- [ ] Review all `Areas/` for anything needing attention
- [ ] Check area standards and KPIs
- [ ] Identify any new tasks or projects

### Review Calendar
- [ ] Review past week for any missed items
- [ ] Review next 2 weeks for upcoming commitments
- [ ] Block time for critical tasks

---

## 3. Get Creative (10 min)

### Review Someday/Maybe
- [ ] Review `GTD-Tasks/3-Someday-Maybe/`
- [ ] Activate any items that are now ready
- [ ] Delete items that no longer resonate

### Review To-Think-About
- [ ] Review `GTD-Tasks/5-To-Think-About/`
- [ ] Schedule thinking time for important items
- [ ] Convert any that are now clear into next actions

### Capture New Ideas
- [ ] Review the week - what emerged?
- [ ] Capture any new ideas or insights
- [ ] Add to appropriate lists

---

## 4. Relationship Check (10 min)

### Marriage Health
- [ ] How was presence with Natalie this week?
- [ ] Any bids for connection missed?
- [ ] Plan one intentional connection for next week
- [ ] Review Gottman 5:1 ratio - how did we do?

### Family
- [ ] Any issues with Matthew or John Francis?
- [ ] Upcoming family events or needs?

---

## 5. Set THE ONE THING (5 min)

### Next Week's Priority
- [ ] What's the ONE THING for next week?
- [ ] Block time for it on calendar
- [ ] Identify potential obstacles
- [ ] Set success criteria

---

## 6. Spiritual Reflection (Optional, 5 min)

- [ ] What am I grateful for this week?
- [ ] Where did I see God working?
- [ ] What virtue do I want to focus on next week?
