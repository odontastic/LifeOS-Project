# Active Projects Dashboard

**Query Definition:** Show all projects where:
- Location: `Projects/Active/`
- `status` = "Active"

**For each project, show:**
- Current next actions (from GTD-Tasks/1-Next-Actions/)
- Waiting-for items (from GTD-Tasks/2-Waiting-For/)
- Project deadline (if any)
- Last updated date

**Use when:** Weekly review or when you want to see the big picture of what you're working on.
