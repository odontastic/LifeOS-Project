# @Computer Actions

**Query Definition:** Show all tasks where:
- `status` = "Next-Action"
- `context` contains "@Computer"

**Sort by:** 
1. Priority (Critical first)
2. Time (shortest first)

**Use when:** You're at your computer and want to see what you can work on.
