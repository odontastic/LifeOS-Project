# Quick Wins (Low Energy, Short Time)

**Query Definition:** Show all tasks where:
- `status` = "Next-Action"
- `energy` = "Low"
- `time` <= "15min"

**Use when:** You're tired but want to make progress. These are tasks you can knock out even when you're running on fumes.
