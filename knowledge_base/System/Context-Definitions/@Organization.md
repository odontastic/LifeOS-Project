# @Organization Context

## Description
Actions focused on organizing, sorting, categorizing, and establishing systems for managing physical or digital items.

## Typical Activities
- Sorting and categorizing items
- Establishing storage systems
- Creating organizational frameworks
- Inventory management
- System setup and implementation

## Related Projects
- Kitchen Countertop Organization (sorting and storage)
- Task Management Implementation (system setup)
- Various organization and system-building projects

## Equipment Needed
- Storage containers and organizers
- Labeling supplies
- Sorting tools
- Documentation materials

## Best Time to Execute
- During organization-focused sessions
- When establishing new systems
- When items need to be sorted and categorized
