## üìù HOW TO USE THIS LIBRARY

### 1. **Start with the "Big Three"**
   - `extract_wisdom` for learning
   - `t_find_blindspots` for planning
   - `create_mermaid_visualization` for clarity

### 2. **Chain Prompts Together** (Fabric Philosophy)
   Example workflow:
   ```
   Article ‚Üí extract_wisdom ‚Üí create_summary ‚Üí write_micro_essay
   ```

### 3. **Add Personal Context**
   Create a `context.md` file (see instructions below) and include it when using prompts for personalized results.
