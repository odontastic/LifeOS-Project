# Look up iCal URLs for calendars
**Objective**: Enable Antigravity to read calendar data on demand.

## Next Actions
- [ ] Go to Google Calendar Settings > Settings for my calendars
- [ ] Select the specific calendar (e.g., Personal, Family)
- [ ] Scroll down to "Integrate calendar"
- [ ] Copy the "Secret address in iCal format"
- [ ] Save the URL to password manager
- [ ] Save these URLs into a secure file (e.g., `System/Calendar_Links.md`) for the agent to use.
